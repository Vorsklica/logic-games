# Гра Input

## Призначення

Input — інтерактивна логічна задача з відкритою відповіддю.

На відміну від Find Next, гра містить лише одну задачу. Користувач вводить відповідь у текстове поле та перевіряє її натисканням кнопки «Перевірити». У разі неправильної відповіді можна повторити спробу необмежену кількість разів.

# Призначення файлів

## config.js

Містить параметри роботи гри.

Не містить жодних даних конкретної задачі.

Приклади параметрів:

- назва гри;
- версія;
- чутливість до регістру;
- автоматичне видалення пробілів;
- максимальна довжина відповіді;
- інші параметри механіки.

---

## content

Папка містить набори даних.

Кожен набір відповідає окремій публікації Telegram.

Опубліковані набори даних не змінюються.

Для нової задачі створюється нова папка:

```
data-002
data-003
...
```

---

## info.js

Містить службову інформацію про набір.

Наприклад:

- id
- title
- created
- author

---

## task.js

Містить умову задачі.

До складу задачі входять:

- текст умови;
- зображення;
- масив правильних відповідей.

Перед перевіркою відповідь користувача нормалізується:

- видаляються пробіли на початку та в кінці;
- переводиться у нижній регістр;
- можуть застосовуватись інші правила згідно config.js.

---

# Інтерфейс

На екрані відображаються:

- назва гри;
- текст задачі;
- ілюстрація;
- поле введення;
- кнопка «Перевірити»;
- повідомлення про результат.

При правильній відповіді відображаються:

- повідомлення «Правильно»;
- відповідний emoji.

При неправильній відповіді:

- повідомлення «Неправильно»;
- можливість повторити спробу.

---

# Клавіатура

Використовується системна клавіатура браузера.

Тип поля визначається HTML (`text`, `number` тощо).

Власна екранна клавіатура не використовується.

---

# Загальні принципи

- усі дані конкретної задачі знаходяться всередині відповідної папки `data-XXX`;
- структура гри повинна бути максимально схожою на інші ігри платформи;
- дизайн повинен використовувати спільні стилі Logic Games;
- у нижній частині сторінки використовується спільний футер платформи.
