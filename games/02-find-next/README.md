## Журнал розробки

### 2026-07-12

Підготовлено архітектуру гри.

Прийняті рішення:

- завдання проходяться послідовно;
- використовується система наборів (`set`);
- опубліковані набори більше не змінюються;
- кожен набір має файл `info.js`;
- сторінка містить мінімальний HTML-каркас;
- гра розробляється поетапно: HTML → CSS → JavaScript;
- для локальної розробки використовується Live Server.
