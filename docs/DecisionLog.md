# Decision Log

## DL-001

### Рішення

Logic Games розробляється як незалежна вебплатформа з підтримкою Telegram Mini Apps.

### Причини

- Telegram є основним способом залучення аудиторії.
- Платформа повинна працювати і без Telegram.
- Це спрощує розробку, тестування та подальший розвиток проєкту.

(залишимо місце, пізніше можна додавати дату)

## DL-002

### Рішення

Для вивчення Telegram Bot API перші приклади реалізуються без використання спеціалізованих бібліотек.

### Причини

Це дозволить краще зрозуміти принцип взаємодії між програмою та Telegram через HTTP-запити.

Використання готових бібліотек буде розглянуто після засвоєння основ.

Рішення №... 08.07.2026
Тема

Telegram Mini App

Прийняте рішення

Вебплатформа Logic Games офіційно використовується як Telegram Mini App.

Основною платформою проєкту залишається Telegram, а вебплатформа виступає середовищем виконання інтерактивних логічних ігор.

GitHub Pages використовується як хостинг Mini App.

Причини
відсутність витрат на власний сервер;
офіційна підтримка Telegram Mini Apps;
можливість використовувати WebApp API;
подальший розвиток без зміни архітектури.

Платформа Logic Games керує не іграми, а публікаціями (posts).
Кожна публікація описується окремим JSON-файлом.
Тип публікації (game, image, video, audio, text тощо) визначає модуль, який виконає її публікацію.
Таким чином бот не залежить від конкретних ігор і може працювати з будь-яким типом контенту.
