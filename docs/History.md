Історія проєкту Logic Games
Передмова

Кожен програмний проєкт має свою історію. Зазвичай вона залишається лише у пам'яті розробників, але з часом багато важливих деталей забуваються. Цей документ покликаний зберегти історію створення Logic Games — не лише технічні рішення, а й ідеї, сумніви, відкриття та етапи розвитку.

Початок

Проєкт Logic Games народився з бажання створити сучасну платформу логічних ігор українською мовою.

Першою ідеєю було створення звичайного вебсайту з логічними задачами та іграми.

Однак уже на початку роботи стало зрозуміло, що сайт сам по собі не є головною метою.

Поступово сформувалося нове бачення.

Головним продуктом має стати Telegram-канал, а вебплатформа повинна виконувати роль інструмента для запуску інтерактивних ігор.

Народження вебплатформи

Було створено репозиторій Logic Games.

Одразу було прийнято кілька важливих рішень.

використовувати Git;
вести журнал розробки;
документувати всі архітектурні рішення;
працювати невеликими завершеними етапами.

Було створено першу вебсторінку та опубліковано її через GitHub Pages.

Вперше сторінка відкрилася не лише на комп'ютері, а й на смартфоні.

Це стало першим підтвердженням правильності обраної архітектури.

Усвідомлення справжньої мети

Під час обговорення майбутнього проєкту стало зрозуміло, що головна цінність Logic Games — не вебсайт.

Головною має бути спільнота навколо Telegram-каналу.

Саме там користувачі отримуватимуть:

нові задачі;
логічні вікторини;
новини;
анонси;
доступ до інтерактивних ігор.

Від цього моменту вебплатформа стала розглядатися як інструмент Telegram, а не як самостійний продукт.

Поява бота

Наступним кроком стало знайомство зі світом Telegram Bot API.

Було створено першого бота.

Під час цієї роботи довелося познайомитися з новими для проєкту технологіями:

Node.js;
npm;
dotenv;
Telegram Bot API;
JavaScript.

Було написано першу програму, яка змогла надіслати повідомлення у Telegram.

Це був перший момент, коли написаний код почав взаємодіяти із зовнішнім світом.

Telegram Mini App

Після цього вебплатформу було інтегровано з Telegram.

Через BotFather було зареєстровано Mini App.

Після кількох перевірок та невеликих труднощів було знайдено потрібне налаштування — увімкнення кнопки запуску Mini App.

Після цього Logic Games вперше відкрилася всередині Telegram.

Саме цей момент можна вважати народженням першої працездатної версії платформи.

На цьому етапі

структуру проєкту;

репозиторій GitHub;
GitHub Pages;
Telegram-бота;
Telegram Mini App;
механізм взаємодії між Telegram і вебплатформою.

Фундамент проєкту було завершено.

Наступний розділ

Наступною великою віхою історії має стати створення першої справжньої логічної гри.

Саме з цього моменту Logic Games почне виконувати своє головне призначення.

9 липня 2026 року — розпочато розробку першої гри платформи Logic Games.
Після обговорення різних варіантів першою грою Logic Games було вирішено зробити логічну гру з перемиканням клітинок. Це буде перша авторська гра платформи, яка стане основою для архітектури всіх наступних ігор.

📙 Історія створення проєкту

09 липня 2026 року

Сьогодні відбувся, на мою думку, один із найважливіших етапів проєкту.

Ми перестали сприймати HTML як саму гру і побудували першу справжню архітектуру застосунку.

З цього моменту логіка гри живе в масиві board, а вебсторінка лише відображає його стан.

Це рішення стане основою не лише для гри «Два кольори», а й для всіх наступних ігор платформи Logic Games.

10 липня 2026 року була завершена перша працездатна версія логічної гри "Два кольори". Вперше було реалізовано повний цикл роботи гри: генерація → взаємодія → перевірка → завершення. Саме ця гра стала основою архітектури майбутньої платформи Logic Games.

# Версія 0.1 — Замкнуто повний цикл платформи

**Дата:** 11.07.2026

## Подія

Завершено перший повний цикл роботи платформи **Logic Games**.

Успішно пройдено весь шлях від створення гри до її публікації та запуску через Telegram.

## Досягнення

- Створено першу гру **«Два кольори 1.0»**.
- Побудовано структуру проєкту Logic Games.
- Підготовлено документацію проєкту.
- Налаштовано середовище розробки VS Code.
- Організовано керування версіями за допомогою Git і GitHub.
- Створено та налаштовано Telegram-бота.
- Створено тестовий Telegram-канал **«Граймонт — Тест»**.
- Розміщено платформу на GitHub Pages.
- Бот успішно опублікував гру на тестовому каналі.
- Гру відкрито через Telegram, повністю пройдено та підтверджено її працездатність.

**Коло замкнулося.**

Платформа Logic Games вперше виконала своє головне призначення — доставила створену гру від розробника до гравця.
