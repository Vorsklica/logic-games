Створення Telegram-бота та Mini App для проєкту Logic Games

Вступ

Telegram дозволяє створювати не лише канали для публікації новин, а й ботів, які можуть взаємодіяти з користувачами та відкривати вебзастосунки (Mini Apps).

У проєкті Logic Games використовується саме така архітектура:

Telegram-канал
        │
        ▼
Telegram-бот
        │
        ▼
Telegram Mini App
        │
        ▼
GitHub Pages

Канал інформує користувачів про новини, бот забезпечує взаємодію, а Mini App містить самі логічні ігри.

Крок 1. Створення бота
Відкрити чат з BotFather.
Виконати команду
/newbot
Ввести назву бота.

Наприклад

Logic Games
Ввести унікальне ім'я
GraimontBot

(обов'язково закінчується словом bot).

BotFather створить бота та видасть BOT TOKEN.

Наприклад

123456789:AA....

Цей токен потрібно зберігати в секреті.

Крок 2. Створення проєкту Node.js

У папці проєкту виконати

npm init -y

Після цього встановити бібліотеки

npm install node-telegram-bot-api
npm install dotenv
Крок 3. Створення файлу .env

У папці проєкту створити файл

.env

та записати

BOT_TOKEN=...
CHAT_ID=...

Файл .env не повинен потрапляти до GitHub.

Для цього у файл .gitignore потрібно додати

.env
Крок 4. Отримання CHAT_ID

Найпростіший спосіб.

Написати будь-яке повідомлення боту.
Відкрити
https://api.telegram.org/bot<ТОКЕН>/getUpdates
У відповіді знайти
chat

і значення

id

Це і є CHAT_ID.

Крок 5. Надсилання першого повідомлення

Створити файл

index.js

У ньому:

підключити dotenv;
прочитати BOT_TOKEN;
прочитати CHAT_ID;
створити екземпляр TelegramBot;
викликати
bot.sendMessage(...)

Після запуску

node index.js

бот надсилає повідомлення у Telegram.

Крок 6. Створення Mini App

Вебплатформа розміщується на GitHub Pages.

Наприклад

https://vorsklica.github.io/logic-games/
Крок 7. Реєстрація Mini App

У BotFather:

/mybots

↓

вибрати свого бота

↓

Bot Settings

↓

Configure Mini App

↓

Enable Mini App

↓

ввести URL GitHub Pages.

Після цього Telegram запам'ятовує адресу вебплатформи.

Крок 8. Увімкнення кнопки запуску

Після реєстрації Mini App потрібно в налаштуваннях бота увімкнути кнопку запуску Mini App.

Саме цей крок ми спочатку пропустили, через що кнопка не з'являлася.

Після ввімкнення кнопки бот відкриває Mini App безпосередньо всередині Telegram.

Крок 9. Перевірка

Після запуску Mini App можна перевірити

Telegram.WebApp.platform
Telegram.WebApp.version
Telegram.WebApp.colorScheme
Telegram.WebApp.initData

Якщо initData існують, значить сторінка працює саме як Telegram Mini App.

Отриманий результат

Після виконання всіх кроків отримано повністю працездатну систему:

Користувач
     │
     ▼
Telegram
     │
     ▼
Logic Games Bot
     │
     ▼
Mini App
     │
     ▼
GitHub Pages

Бот відкриває вебплатформу безпосередньо всередині Telegram, а вебсторінка отримує доступ до Telegram WebApp API.
