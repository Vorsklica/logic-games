# Vision

## Місія

Logic Games — це проєкт, центром якого є Telegram-канал, а вебплатформа є інструментом для інтерактивних логічних ігор.

## Основна ідея

Користувач регулярно відкриває Telegram-канал, щоб отримати нову логічну задачу, головоломку, вікторину або інтерактивну гру.

Прості формати (текстові задачі, опитування, вікторини) залишаються всередині Telegram.

Інтерактивні ігри відкриваються на вебплатформі Logic Games.

## Принципи

- Telegram — головний канал взаємодії з аудиторією.
- Вебплатформа — інструмент для інтерактивних ігор.
- Ігри повинні працювати і через Telegram Mini Apps, і у звичайному браузері.
- Нові ігри мають легко додаватися без зміни архітектури платформи.

Logic Games поступово переходить від окремої вебсторінки до повноцінного Telegram Mini App. Користувач повинен запускати гру безпосередньо всередині Telegram, не залишаючи застосунок. Саме цей режим роботи розглядається як основний для майбутнього розвитку платформи.
